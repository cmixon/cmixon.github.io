    **Note: Do not use brackets even if reporting on <20 metaphases**

    • Interpretation field
  • Infertility macro pb (recommends studying spouse) if reason for referral is recurrent miscarriages or other history regarding reproductive issues

Oncology

    Karyotype field:
  • ISCN correct including sex chromosome designation and number range for composite
  • Composite karyotype follows rules for clonality (3 cells for losses, 2 for structural abnormalities or gains)
  • Communication to cytogenetics director written in Karyotype field
  • If No Growth, put "No metaphase cells for analysis" macro here.

    • Interpretation field
  • Long-term culture macro in APVX for any normal karyotype with study of long term culture ("Chrome Analysis long term culture only"); also at Other Macros section
  • PB failure (< 5 metaphases) macro ("Chrome-Analysis -Blood failure")
  • Correct abnormal macro chosen
  • Abnormal writeup started if not for Dr. Mixon
  • If failure AND a COMPASS case with no pending FISH, email Pathology Support

    • Comments field
  • GEN COMM required for any case reporting at least 1 metaphase
  • Reflex testing added per request (usually SNP/microarray macro)
  • Linked report if patient's spouse or parent is being or has been studied
  • NeoTrack macro as applicable for previous cases
    PREV ABNOR 1,2,3 PREV NOR 1,2 in APVX
    or Generic from Other Macros section
  • "Previous reports" macro (normal or abnormal) for cases not in NeoTrack (see Other Macros section)

    • Test Detail section
  • Number of cells counted and analyzed correct; should match number in [ ] in karyotype
  • Banding level if different from default
  • Most photogenic karyotype chosen
  • Label on report karyotype correct
  • Study sheet uploaded
  • Culture failure sheet uploaded
  • Diagnosis Type dropdown box (Report Options) answered
